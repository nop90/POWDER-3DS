Powder2X version 0.1
====================

Port of the Roguelike Powder release 104 (see http://www.zincland.com/powder).

Controls
========

In Main Menu
-------

Joystick - Select option
A        - Select

In Game
-------

Joystick - Rotate
A	 - Do action/Fire
B        - Cancel/Spell
L	 - Bind action
R        - Map
Select   - Inventory
Start    - Menu

It's best to do the tutorial to find out the controls in full.
