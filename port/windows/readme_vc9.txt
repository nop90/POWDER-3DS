POWDER Visual C++ 2008 Express Edition (VC9) Solution File

Quick guide:

- Download and install Microsoft Visual C++ 2008 Express Edition
- Download and extract libSDL Win32 development libraries
- Add SDL include and library directories to VC configuration
- Download and extract POWDER source code
- Extract this archive to POWDER source code directory
  (all files should be in <POWDER-Source>\port\windows)
- Open powder_vc9.sln, VC should start up
- Choose configuration (Debug|Release)
- Build POWDER
- Copy SDL.dll to build directory <POWDER-Source>\port\windows\(Debug|Release)
- Start Powder.exe

If you want to clean up after you, start prebuild_clean.bat
